Writing Samples

PowerShell Script Documentation

Wrote GitHub documentation for an open-source PowerShell script coded by me.

  • Simple installation instructions to support users unfamiliar with GitHub or PowerShell
  • Concise features list outlining the important functions of the program
  • Friendly, spoken-voice style that guides the reader using second-person pronouns

MLA Formatting Guide

Created my own guide for teaching middle and high school students how to format their essays in MLA.

  • Simple, concise language to help students new to English or computers follow along
  • Step-by-step instructions to help students quickly find information they need
  • Screenshots of Google Docs with highlighted sections to show students where on their screen to click
  • Model MLA examples for students to compare their work against

Wiki Netplay Guide

Rewrote the official wiki's page on how to play Pokemon Close Combat online. Improvements include:

  • New organization to help users find what they're looking for quickly
  • Reference images with highlighted sections to help users follow technical instructions
  • Twice as much information devoted to teaching online interface
  • Text markup to help users see what they're supposed to click
  • Simpler language to help users understand instructions
  • Rewritten text in line with Microsoft's Writing Style Guide

AI Logic Documentation

Converted gaming content created haphazardly by many contributors into organized documentation. Improvements include:

  • Concise writing that, despite adding topics, shortens document by more than 1,000 words
  • Table of contents with information organized into discrete topics to help users navigate
  • Embedded hyperlinks to help users quickly jump to relevant info
  • Tables to help users visualize AI decisions more clearly (see example below)
  • Consistent language that always refers to each concept by the same name
